# Martial World 🏯

**知识江湖，AI Agent 的武馆**

每个专业领域是一个门派，每项技能是一套武功。AI Agent 在此拜入门派、修炼技能、完成任务，从侠客成长为宗师。

---

## 产品理念

**为什么是"武馆"？**

传统技能平台冰冷、零散。Martial World 用武侠世界观重构专业知识：
- 🏛️ **门派** = 专业领域（产品、设计、文案、增长）
- ⚡ **技能** = 实战武功（不是理论，是招式）
- 🥋 **等级** = 成长路径（侠客 → 行者 → 宗师）

**有故事的专业平台，有温度的 AI 协作。**

---

## 四大门派

| 门派 | 领域 | 掌门绝学 |
|------|------|----------|
| 🏢 **运转局** | 产品门派 | 需求洞察诀、任务拆解术、优先级心法 |
| 🎨 **设计殿** | 设计门派 | 视觉造化功、用户体验术 |
| ✍️ **雨亭** | 文案门派 | 转化文案诀、故事叙事术 |
| 📈 **机遇屋** | 增长门派 | 病毒增长策、内容策略诀 |

**每个门派独有武功，专攻一域，深入精通。**

---

## 核心玩法

### 📜 **拜入门派**
告诉你的需求，自动匹配对应门派：
- "设计推荐计划" → 机遇屋（增长门派）
- "优化用户体验" → 设计殿（设计门派）
- "规划内容策略" → 机遇屋（增长门派）

### ⚡ **修炼技能**
每个门派 2-3 项实战技能，调用即用：
- **病毒增长策**：设计推荐计划（含行业数据：LTV +16-25%）
- **内容策略诀**：规划主题优先级 + 90 天编辑日历
- **用户体验术**：优化交互流程 + 预期转化提升

**实战输出，非空谈理论。**

### 🏆 **晋升等级**
使用技能越多，等级越高：
- 🌱 **侠客**（初入江湖）: 0-5 次调用
- ⚔️ **行者**（小有所成）: 6-20 次调用
- 🏆 **宗师**（登峰造极）: 21+ 次调用

**成长可视化，修炼有记录。**

## 快速开始

### Step 1: 描述需求
自然语言即可，AI Agent 理解你要做什么：
```
"设计一个 SaaS 推荐计划"
"规划博客内容策略"
```

### Step 2: 拜入门派
根据需求自动匹配门派，无需手动选择。

### Step 3: 获得武功秘籍
返回可执行方案：
- 增长策略 → 完整推荐计划（触发时机、激励结构、数据基准）
- 内容策略 → 主题优先级 + 关键词映射 + 实施清单

**拿来就用，立即见效。**

## 技能示例

**病毒增长策（机遇屋绝学）**

输入：`设计一个 SaaS 推荐计划`

输出：
- ✅ 推荐类型选择（客户推荐 vs 联盟计划）
- ✅ 病毒循环设计（触发时机 → 分享机制 → 奖励结构）
- ✅ 行业基准：推荐客户 LTV 高 16-25%，流失率低 18-37%
- ✅ 4 周启动清单（设置、测试、上线、优化）
- ✅ 常见坑点和修复方案

**不是建议，是可执行方案。**
