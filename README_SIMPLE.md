# Martial World

**AI Agent 专业技能平台**

告诉你的需求，AI Agent 调用专业技能帮你完成工作。不用到处找方法，一站搞定专业任务。

---

## 能帮你做什么

| 需求 | 输出 |
|------|------|
| 设计推荐计划 | 完整方案 + 数据基准 + 实施清单 |
| 规划内容策略 | 优先级主题列表 + 编辑日历 + 关键词映射 |
| 优化用户体验 | 具体改进建议 + 预期效果 |
| 撰写营销文案 | 转化导向的文案 + A/B 测试方案 |

**做实际工作，不是给建议。**

---

## 核心玩法

**4 大门派，对应 4 类专业领域：**

| 门派 | 领域 | 能做什么 |
|------|------|----------|
| 🏢 运转局 | 产品 | 需求洞察、任务拆解、优先级排序 |
| 🎨 设计殿 | 设计 | 视觉设计、用户体验优化 |
| ✍️ 雨亭 | 文案 | 转化文案、故事叙事 |
| 📈 机遇屋 | 增长 | 病毒推荐、内容策略 |

**告诉需求 → 匹配门派 → 调用技能 → 完成任务**

---

## 快速开始

### Step 1: 告诉 AI Agent 你的需求

自然语言描述即可：
- "设计一个 SaaS 推荐计划"
- "规划博客内容策略"
- "优化结账流程体验"

### Step 2: 自动匹配门派

AI Agent 根据需求自动匹配对应门派，无需手动选择。

### Step 3: 获得可执行输出

返回具体方案：
- 增长策略 → 推荐计划设计（含行业数据：LTV +16-25%）
- 内容策略 → 主题优先级列表 + 90 天日历
- UX 优化 → 具体改进点 + 预期转化提升

**直接可用，不是空泛框架。**
